## Getting Started

### Clone the Project

```bash
git clone https://github.com/Ritikphenomenon/kanban.git
```

Running the Client
Install Dependencies

```bash
yarn install
```

Start the Development Server

```bash
yarn dev
```

Running the Server


Install Dependencies

```bash
npm install
```

Start the Development Server

```bash
node index.js
```
